# 📱 微积分出题软件 - 移动端使用说明

## 功能特点

✅ **完全响应式设计** - 完美适配手机、平板和电脑  
✅ **PWA支持** - 可添加到主屏幕作为应用使用  
✅ **触屏优化** - 更大的按钮和输入框，适合手指操作  
✅ **流畅动画** - 优秀的视觉反馈和用户体验  
✅ **离线可用** - 一旦加载，无需网络也能使用题库  

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务器

```bash
npm start
```

### 3. 在手机上访问

1. 确保电脑和手机连接同一WiFi
2. 查找电脑IP地址（Windows: `ipconfig`, Mac/Linux: `ifconfig`）
3. 在手机浏览器访问 `http://[电脑IP]:3000`

### 4. 添加到主屏幕（PWA）

**iPhone/Safari:**
- 点击分享按钮 ↗️
- 选择「添加到主屏幕」
- 命名为「微积分出题」

**Android/Chrome:**
- 点击菜单 ⋮
- 选择「添加到主屏幕」
- 确认添加

## 使用方法

1. 🎯 **选择知识点** - 从下拉菜单选择练习的题型
2. 🔢 **设置数量** - 选择要生成的题目数量（1-20题）
3. ✨ **生成题目** - 点击按钮生成随机题目
4. 📖 **查看答案** - 点击「查看答案与解析」显示详细解答

## 涵盖知识点

- 📐 函数与极限
- 📈 导数
- 📉 导数的应用
- 📝 不定积分
- 📋 定积分
- 📐 定积分的应用
- 🧩 微分方程

每个知识点包含10道精选题目，配有详细解答！

## 技术说明

- 前端：原生HTML/CSS/JavaScript（无需框架）
- 后端：Node.js + Express
- 题库：7个模块，70道题目
- 完全开源，可自由扩展

## 扩展题库

编辑 `questionBank.js` 文件，按照现有格式添加新题目即可！

微积分出题软件 - 使用说明

快速开始：

1. 安装依赖：
   打开命令行，进入此目录，运行：
   npm install

2. 启动服务器：
   npm start

3. 使用应用：
   在浏览器中打开 http://localhost:3000

包含的知识点：
- 函数与极限
- 导数
- 导数的应用
- 不定积分
- 定积分
- 定积分的应用
- 微分方程

每个知识点包含10道题目，配有详细答案和解析。

API接口：
- GET /api/topics - 获取所有知识点
- GET /api/questions/random?count=5&topic=函数与极限 - 随机生成题目
- GET /api/questions/:topic - 获取指定知识点的所有题目
- GET /api/question/:id - 获取指定ID的题目
